Sobre Nós

O Planeja festas é um software criado para facilitar a organização de festas para aqueles que precisam organizar um evento simples mas não sabem a quantidade a comprar de cada coisa.

Ele permite que você cadastre novas festas e visualize as antigas que você já criou. Cada festa tem um nome, um número de convidados e um cardápio. O PlanejaFesta faz o trabalho duro para você: calcula a quantidade de cada item do seu cardápio que deve ser comprada.

Além de tudo, o PlanejaFesta ainda conta com um blog onde você pode obter dicas de organização de festas criadas por especialistas no assunto. Não sabe qual o melhor dia para fazer uma festa? Pesquisa no blog! Não sabe qual o melhor local para a realização do seu evento? O blog tem a resposta!
